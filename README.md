# Employee Management System

## Project Overview

This project is a JavaScript-based Employee Management System that demonstrates object-oriented programming (OOP) concepts such as classes, inheritance, and method overriding. It models employees and managers within a company and displays their roles in a structured format.

## Technologies Used

* JavaScript (ES6)
* HTML

## How to Run the Project

1. Open the `index.html` file in your browser.
2. Open the browser’s developer console (Right click → Inspect → Console).
3. View the employee directory output.

## Features

* Create employees and managers using classes
* Use inheritance with `extends` and `super()`
* Override methods for different roles
* Store employees in a company system
* Display a formatted employee directory
